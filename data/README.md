# Data Sources

**No raw data files are stored in this directory.**
All source data remains at its original location. Scripts access it via `scripts/config.py`.

---

## Primary Data Sources

### 1. AEB Individual Survey Responses

- **File:** `historical aeb data - malone.xlsx`
- **Sheet:** `Data`
- **Header:** None (no header row — columns are positional)
- **Rows:** ~49,339 individual survey responses
- **Path configured in:** `scripts/config.py` → `AEB_FILE`

**Column Map (0-indexed):**

| Index | Name | Description |
|-------|------|-------------|
| 0 | Month | 1–12 |
| 1 | Year | 2015–2024 |
| 2 | Q1_raw | Raw Q1 response |
| 3 | Q2_raw | Raw Q2 response |
| 4 | Q3_raw | Raw Q3 response |
| 5 | Q4_raw | Raw Q4 response |
| 6 | Q5_raw | Raw Q5 response |
| 7–8 | (additional raw) | Additional response fields |
| 9 | Q1_idx | Q1 indexed value |
| 10 | Q2_idx | Q2 indexed value |
| 11 | Q3_idx | Q3 indexed value |
| 12 | Q4_idx | Q4 indexed value |
| 13 | Q5_idx | Q5 indexed value |
| 14–15 | (additional idx) | Additional indexed fields |
| 16 | AEB_raw | Raw AEB composite |
| 17 | ICC_raw | Raw Current Conditions |
| 18 | IFE_raw | Raw Future Expectations |
| 19 | AEB | AEB index (col T in method docs) |
| 20 | ICC | Index of Current Conditions |
| 21 | IFE | Index of Future Expectations |

**Notes:**
- Individual future expectations for U_quant = mean of Q2_idx, Q3_idx, Q4_idx per respondent
- AEB (col 19) is the primary measure used in VUCA formula

### 2. Word Cloud Open-Ended Text Responses

- **Location:** `Word Cloud Data/YYYY/word cloud - YY-MM.xlsx`
- **Coverage:** October 2015 – present (~108 files)
- **Path configured in:** `scripts/config.py` → `WORD_CLOUD_DIR`

**File naming convention:** `word cloud - YY-MM.xlsx` (2-digit year, 2-digit month)

**Column structure:** To be confirmed on first load via `00_load_data.py`.
After confirmation, update this table:

| Column | Name | Description |
|--------|------|-------------|
| TBD | text_response | Open-ended farmer text response |

- Files use various encodings; openpyxl handles this automatically

## Cleaned Analysis Datasets (output/)

These files are generated by `scripts/00_load_data.py`:

| File | Description |
|------|-------------|
| `output/aeb_monthly.parquet` | Monthly aggregates: mean AEB, variance of IFE, N respondents |
| `output/text_monthly.parquet` | Monthly text: list of responses per month, token counts |

Generated by downstream scripts:

| File | Description |
|------|-------------|
| `output/quant_measures.parquet` | V_quant, U_quant, A_quant, C_fallback per month |
| `output/text_measures.parquet` | V_text, U_text, C_text, A_text per month |
| `output/vuca_monthly.parquet` | Final standardized V, U, C, A scores + all sub-measures |
