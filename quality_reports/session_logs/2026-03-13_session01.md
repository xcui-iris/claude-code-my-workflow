## Key Decisions Made

| Decision | Rationale | File |
|----------|-----------|------|
| U_quant = inverted IFE (not cross-sectional variance) | Cross-sectional variance captures disagreement, not uncertainty; convergent validity r went from +0.22 to −0.758 | `quality_reports/specs/2026-03-13_u-quant-decision.md` |
| Baseline = 2016–2019 | Pre-COVID, post-election; includes trade war to avoid artificially low uncertainty baseline | `CLAUDE.md` |
| C_score uses LDA when available, C_fallback otherwise | LDA provides richer complexity signal; fallback (ICC-IFE spread SD) used when sklearn unavailable | `scripts/03_combine_standardize.py` |
| AEB column assignment: col 0 = Year (2-digit), col 1 = Month | Confirmed from date range diagnostic (15-01 → 25-12) | `scripts/00_load_data.py` |
| Word Cloud text column = index 1 | Auto-detected by max string length; confirmed: "GAS PRICES, WIND TURBANS." was col 0 header misread | `scripts/00_load_data.py` |
